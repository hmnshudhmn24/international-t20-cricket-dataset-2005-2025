# International T20 Cricket Dataset (2005–2025)

A detailed ball-by-ball dataset covering **4,801 T20 International matches** played between **2005 and 2025**.
This dataset includes delivery-level events, runs, wickets, extras, player actions, and complete match metadata—ideal for cricket analytics, machine learning, and research.

## Features
- 4,801 T20I matches
- Ball-by-ball delivery data
- Runs, wickets, extras, dismissals
- Batting, bowling, and fielding details
- Match info: teams, venue, date, toss, result

## Format
Available in **JSON** and **YAML** formats.
